# 📝 Note Manager

A simple and efficient **Note Management Web Application** that allows users to create, read, update, and delete notes. The application helps organize notes using categories and provides powerful filtering, searching, and sorting options for better productivity.

---

# 📌 Project Overview

**Note Manager** is a CRUD-based web application designed to manage personal and professional notes efficiently.
Users can easily create notes, categorize them, filter them by status, search by subject, and sort them in an organized way.

The goal of this project is to provide a **clean and intuitive interface** to manage notes effectively.

---
# 🚀 Features

### 1.✏️ CRUD Operations

* Create new notes
* Read/view existing notes
* Update notes
* Delete notes

### 2.🔍 Search Notes

* Search notes by **subject**
* Quickly find specific notes

### 3.📂 Category Organization

Notes are organized into categories:

* Work
* Personal
* Study

Users can easily view notes based on their category.

### 4.🗂 Status Filtering

Filter notes based on their status:

* **All** → Show all notes
* **Completed** → Show completed notes
* **Pending** → Show pending notes

### 5.🔃 Sorting

Sort notes by **subject**:

* **Ascending (ASC)**
* **Descending (DESC)**

### 📋 Organized Listing

All notes are displayed in a structured and easy-to-read format.

# 📦 Installation & Setup

Follow these steps to run the project locally.

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/sandhya1833/mern-note-manager.git
```

### 2️⃣ Navigate to the Project Folder

```bash
cd note-manager
```

### 3️⃣ Install Dependencies

Frontend:

```bash
cd frontend
npm install
```

Backend:

```bash
cd backend
npm install
```

# 📖 Usage

1. Create a new note.
2. Assign a **category** (Work, Personal, Study).
3. Add a **status** (Completed / Pending).
4. Search notes using the **subject search bar**.
5. Filter notes using **status buttons**.
6. Sort notes **ascending or descending by subject**.
